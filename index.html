Arcana Auth: AWS Cognito Login with Custom UI!

This simple HTML/CSS/JS Web3 app example to demonstrate how a Web3 app can use custom login UI and allow user onboarding via AWS Cognito or passwordless.

It uses Parcel bundler.

Use model:

  1. Log in via Cognito or passwordless.
  2. Get information about the user, account, login status.
  3. Perform wallet operations using the Arcana wallet UI
  4. Logout
